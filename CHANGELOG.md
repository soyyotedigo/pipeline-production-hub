# Changelog

All notable recruiter-facing changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project currently uses a simple `v0.x` portfolio release cadence.

## [0.1.0] - 2026-04-24

### Added

- portfolio-oriented README refresh with architecture summary, visual walkthrough, operational evidence, and recruiter-friendly positioning
- fixed interview demo script with seeded credentials, entities, and a required version/file flow
- public architecture screenshots under `docs/images/`
- DCC integration examples under `examples/dcc/`
- smoke test suite covering core API domains end to end

### Changed

- curated public sync flow for keeping `pipeline-production-hub` aligned with `pipeline-production-hub-dev`
- public-facing docs for storage, DCC integration, testing workflow, and quality tooling
- API example in the README to better reflect seeded production-style data instead of placeholder-only CRUD

### Notes

- this release is intentionally portfolio-focused: it packages the current backend, docs, demo flow, and public evidence into a recruiter-ready snapshot rather than introducing a new product milestone
